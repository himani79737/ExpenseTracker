💸 SpendSmart – Expense Tracker App

A modern and responsive Expense Tracker Web Application built with React that helps users manage daily expenses, track spending patterns, and monitor budgets efficiently.

🚀 Features
➕ Add new expenses with title, amount, category, date, and notes
🗑️ Delete expenses with confirmation modal
🔍 Filter expenses by category and month
📊 Visualize spending using Pie Chart (category-wise) and Bar Chart (monthly)
💰 Set and track monthly budget with progress indicator
⚡ Data persistence using localStorage (no backend required)
🔔 Toast notifications for user actions
📱 Fully responsive and modern UI
🛠️ Tech Stack
#Frontend: React, JavaScript, HTML, CSS
#State Management: React Hooks (useState, useEffect, useMemo)
#Charts: Recharts
#Storage: Browser localStorage
